Java Script - ściągawka

Jak dodać obsługę zdarzenia klinięcia myszki

Dodaj atrybut/zdarzenie onclick do elementu i przekarz sterowanie do funkcji. 
        <button id="liczba" onclick="dodaj()"/>
        <script>
          function dodaj() {
           // ...
          }

        </script>

Jak stworzyć funkcje

        <script>
          function dodaj() {
           // ...
          }

        </script>

Jak używać funkcji

        <script>
          function dodaj() {
           // ...
          }

           dodaj()
        </script>

Funkcja z parametrami

      <script>
        //Definicja
        function poleProstokatu(a,b) {
          return a*b;
        }


        // Użycie
        poleProstokatu(5,8);
     
      </script>

Pobieranie wartości pola tekstowego

        <input id="liczba"/>
        <script>
          var liczbaText = document.getElementById("liczba").value;
          var liczba = parseInt(liczbaText);

        </script>

Jak wstawić kod HTML na stronę

        <div id="pole"/>
        <script>
          document.getElementById("pole").innerHTML = "Moj kod";
        </script>

Jak zmienić klasę elementu

        <div id="pole"/>
        <script>
          document.getElementById("pole").className = "pogrugiony";
        </script>

Jak odczytać klasę elementu

        <div id="pole"/>
        <script>
          var klasa = document.getElementById("pole").className
          alert("Element pole ma klase: "+klasa);
        </script>

Pętla while

Schemat: 
        while (<warunek>) {
          <kod>
        }
Przykład: 
  <script>
  var i = 0;
  while (i<10) {
    alert(i);
    i++;
  }
  </script>

Warunki (if-y)

  <script>
  var wiek = parseInt(prompt("Podaj wiek"));
  if (wiek>=18) {
    alert("Możesz wejść.");
  } else {
    alert("Jesteś za młody.");
  }
  </script>
